密钥交换协议kyber算法原理（基于LWE困难问题）： 
简单理解LWE问题：
想象一个“嘈杂”的线性方程组：
=你有一个公开的矩阵 A，你有一个秘密的向量s。
计算 A×s=t，A×s=t（如果没有错误，这很简单）。
但关键来了：在实际问题中，你会得到的是 A×s+e=t
A×s+e=t，其中e是一个很小的、随机的错误向量。
LWE问题就是：给定公开的A和 t，在存在小错误e的情况下，
恢复出秘密向量 s 是极其困难的。
